# Required Assets

Please add the following images to this `public/` directory to complete the visual setup:

1. **Hero Background**
   - File: `hero-placeholder.jpg`
   - Description: A high-quality, dark, cinematic DayZ screenshot for the homepage hero section.

2. **Map Thumbnails**
   - Files:
     - `maps/deer-isle.jpg`
     - `maps/namalsk.jpg`
     - `maps/chernarus.jpg`
   - Description: Screenshots or map previews for each server card. Ensure they are dark/atmospheric to match the theme.

3. **Join Guide Images**
   - File: `launcher-guide.jpg`
   - Description: Screenshot of the launcher for the join guide.

4. **Grid Pattern**
   - File: `grid.svg` (Optional, CSS fallback is used but SVG is better)
   - You can use a simple SVG pattern generator online.

## Notes

- All images should be optimized for web (WEBP or JPG, under 200KB if possible).
- The `Home` page hero section has a grayscale overlay, so colorful images will be muted automatically.
